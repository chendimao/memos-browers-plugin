# 编辑页分层重设计与外层去圆角实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**功能目标：** 去掉外层 `memos-extension` 的圆角，并重设计编辑页的层次样式，让编辑框与工具区、底板背景有明确区分，同时保持现有交互和布局结构不变。

**技术方案 / 架构说明：**
本次只做样式层面的最小重设计，不改动 Vue 组件结构和交互逻辑。核心思路是把编辑页重构成三层视觉层级：外层浅雾灰底板、次级功能面板、主编辑卡片。外层 `memos-extension` 改为无圆角矩形弹窗，`editor-wrapper` 改为最亮的主编辑区域，通过背景、描边、阴影和焦点态增强可辨识度。

**涉及文件：**
- 修改：`/Users/chendimao/WWW/memos-browers-plugin/.worktrees/feat-figma-restyle-pages-20260227/src/App.vue`
- 可能更新导出：`/Users/chendimao/WWW/memos-browers-plugin/export/`

**潜在风险：**
- 颜色层级调整后，暗色主题可能出现对比不一致，需要检查现有 dark 样式是否被覆盖。
- 工具栏与编辑卡之间的间距、阴影如果过强，可能让 popup 在小尺寸下显得拥挤。
- 外层去圆角后，若边界阴影保留过重，可能与浏览器 popup 边缘冲突。

**步骤任务列表：**

### 任务 2：实现外层去圆角与三层分层风格

**步骤任务列表：**
1. 去掉 `memos-extension` 的圆角，保留矩形弹窗感。
2. 将编辑页底板改成浅雾灰背景，形成与编辑主卡的明度差。
3. 把 `editor-wrapper` 改成最亮的主编辑卡片，增加更清晰的边框和阴影。
4. 让 `content-editor` 背景与编辑卡协同，而不是继续和周围容器融在一起。
5. 调整 `stats-panel` 为更弱的底栏层次，避免与正文区混为一体。

### 任务 3：重做工具区与操作区层次

**步骤任务列表：**
1. 将 `toolbar` 改成次级功能面板，弱于编辑卡但强于页面底板。
2. 调整 `action-tools`、标签选择、可见性选择、按钮区的背景和边框层次。
3. 保持提交按钮为最强强调色，不改动功能逻辑。
4. 检查小尺寸 popup 下工具区换行后的视觉完整性。

### 任务 4：验证与导出

**步骤任务列表：**
