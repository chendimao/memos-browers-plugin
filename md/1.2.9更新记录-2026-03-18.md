# 1.2.9 更新记录

## 发布概览

本版本主要聚焦于跨浏览器分发与部署能力补强，新增更多可直接交付的构建产物，并完善发布流程。

## 更新内容

- 新增 Chrome 与 Firefox 双生产包构建能力，方便分别加载和分发。
- 补齐 Firefox 所需的扩展 ID 与数据权限声明，提升 Firefox 兼容性。
- 新增 Safari 临时安装测试打包能力，便于在 macOS Safari 中进行本地测试。
- 新增 Web 构建产物，支持导出 `dist/web` 并自行部署到静态服务器。
- 新增 GitHub Releases 自动发布流程，可自动上传 Chrome / Firefox / Safari / Web 四类发布包。

## 发布资产

本次发布会包含以下安装 / 部署产物：

- Chrome 离线安装包
- Firefox 离线安装包
- Safari 临时安装测试包
- Web 静态部署包
